## Best Practices

- You can find the exb client code at ~/Code/exb/client
- Read the doc: https://developers.arcgis.com/experience-builder/guide/getting-started-widget/
- For widget example, see widgets in the official arcgis-experience-builder-sdk-resources repo, find it either there ~/tmp/arcgis-experience-builder-sdk-resources/widgets or directly on github https://github.com/Esri/arcgis-experience-builder-sdk-resources

## UI Best Practices

When building UI for arcgis-experience builder widgets, follow these guidelines:

- Use jimu-ui or calcite components, favours using jimu-ui
- For styling try to prioritize using jimu-ui built in bootstrap classes, see https://developers.arcgis.com/experience-builder/storybook/?path=/docs/theme-theme-references-css-utility-classes--docs
- For colors integrate with arcgis-experience-builder and jimu-ui theming system, see https://developers.arcgis.com/experience-builder/storybook/?path=/docs/theme-theme-references-css-variables--docs
- For icons prioritize jimu-ui icons, see https://developers.arcgis.com/experience-builder/storybook/?path=/docs/icons-list-of-icons--docs
